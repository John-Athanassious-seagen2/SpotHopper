Figma Prompt (Strategic, Not AI-Fluff)

Design a modern product marketing website for a restaurant growth platform called “SpotHopper.”

This is not a restaurant website. It is a SaaS product website that sells to restaurant owners.

Tone:
Confident. Direct. Revenue-focused. Not playful. Not startup-bro.

Visual Direction:

Deep navy or charcoal base

Electric blue accent

Clean grid system

No overused gradient blobs

Structured layout with strong hierarchy

Crisp typography with tight spacing

Data-forward aesthetic

Subtle motion cues (hover, micro interactions)

Homepage Structure:

Hero:
Headline: “Own Your Restaurant’s Revenue.”
Subheadline explaining websites, online ordering, catering, marketing automation in one system.
Primary CTA: “Book a Demo”
Secondary CTA: “See How It Works”
Right side: clean dashboard UI mock showing revenue, orders, catering leads.

Social Proof:
Restaurant logos strip

Problem Section:
Short sharp copy explaining:

Third-party dependency

Lost catering revenue

Poor website conversion

Solution Section:
Feature blocks:

Website Builder

Online Ordering

Catering Management

Automated Marketing

CRM

Reporting Dashboard

Each feature should have:

Icon

2–3 line description

Supporting UI mock snippet

Results Section:
Large metric cards:

+32% Direct Orders

+48% Catering Inquiries

99% Customer Retention

How It Works:
Simple 3-step process

Case Study Preview:
Restaurant success story card

Pricing Section:
Three tiers
Clear differentiation
Revenue-focused descriptions

Final CTA:
Strong, bold banner

Footer:
Enterprise-level structure

Desktop 1440px

Tablet and mobile

Full component system

Button variants

Card variants

Form components

Organized design system page

Cartoon illustrations
